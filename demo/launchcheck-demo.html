LaunchCheck

Proof before you ship.

A local QA watcher for vibe-coded ZIP and HTML builds.

1
Drop in a buildZIP or standalone HTML.
2
LaunchCheck inspects itStructure, syntax, browser behavior, and QA contract.
3
Read the evidenceHTML report plus JSON details.
4
Paste the repair packetGive the next AI run clear fixes.
Public release safety: it does not self-update, install builds, or run random executables.
